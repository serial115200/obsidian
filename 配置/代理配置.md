# 启用代理

Obsidian 第三方插件库访问异常，此时需配置代理访问。

Obsidian 基于 Electron 开发，所以在快捷方式的 "目标" 项添加以下参数即可启用代理。

```bash
--proxy-server="http://<ip>:<port>"
```

1. 右键点击 Obsidian 的快捷方式，选择"属性"
2. 找到"目标"项，在原有路径后添加代理参数

假设代理服务器地址为 `127.0.0.1:7890`，则完整的目标路径可能如下：

```
"C:\xxxxx\xxxxx\Obsidian\Obsidian.exe" --proxy-server="http://127.0.0.1:7890"
```
